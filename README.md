# screen-time-analysis

The following repo uses R to look at the relationship between social media use and the sleep habits of 500 people from survey conducted in Singapore.

https://www.kaggle.com/datasets/globalmediadata/socialmediausage-sleepdata-sg?select=SocialMediaUsage_SleepLatencyAnalysis_Singapore.csv

How to Run this Analysis:
Clone this repository

Open screen-time-analysis.Rproj

Open CS544Final\_Duran.Rmd and click Knit
